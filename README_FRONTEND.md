# 前端应用 - Vue.js 智能笔记编辑器

## 功能特性

### 1. 笔记管理
- **笔记列表**：侧边栏显示所有笔记
- **过滤搜索**：按类型过滤和关键词搜索
- **实时加载**：点击刷新按钮重新加载笔记

### 2. AI 分类与处理
- **粘贴处理**：自动 AI 分类判断是否为笔记
- **类型识别**：自动识别笔记类型（待办、知识、灵感等）
- **可信度显示**：显示 AI 分类的置信度

### 3. 编辑功能
- **三视图编辑**：
  - 原始内容：查看未处理的原始内容
  - AI 整理版：查看 AI 组织整理的 Markdown
  - 编辑：修改标题、类型、补充内容
- **富文本支持**：支持 Markdown 格式

### 4. 新建笔记
- 手动输入标题和内容
- 选择笔记类型
- AI 自动整理和组织

## 使用方式

### 打开应用
在浏览器中打开 `index.html` 文件：
\`\`\`bash
# 简单方式：直接在浏览器中打开文件
open index.html

# 或使用本地服务器
python -m http.server 8000
# 访问 http://localhost:8000/index.html
\`\`\`

### 基本操作

1. **加载笔记**：点击 🔄 按钮加载所有笔记
2. **新建笔记**：点击 ➕ 按钮创建新笔记
3. **粘贴处理**：点击 📋 按钮粘贴内容，AI 自动分类
4. **浏览笔记**：点击左侧笔记列表查看详细内容
5. **编辑笔记**：在编辑标签页修改内容

## API 交互

前端与后端通过 REST API 通信：

- `POST /api/classify-content` - 分类内容
- `POST /api/organize-content` - AI 整理内容
- `POST /api/save-note` - 保存笔记
- `GET /api/notes` - 获取所有笔记
- `GET /api/notes/<id>` - 获取单个笔记
- `PUT /api/notes/<id>/edit` - 编辑笔记
- `DELETE /api/notes/<id>` - 删除笔记

## 配置

在 `index.html` 中修改 API 基础 URL：

\`\`\`javascript
const API_BASE = 'http://127.0.0.1:5001/api';
\`\`\`

应用使用现代深色主题，包含以下主要颜色：

- **主色**：紫色 (#7C3AED)
- **副色**：粉红色 (#EC4899)
- **背景**：深灰 (#0F172A)
- **表面**：浅灰 (#1E293B)
- **文本**：明亮 (#F1F5F9)
